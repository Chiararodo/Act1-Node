API Teatro BA

API REST de obras de teatro en Buenos Aires.

Endpoints

  • Teatros
  • Obras
  • Usuarios

Probá GET, POST, PUT, DELETE con Postman.

Filtros disponibles

  • Obras: ?q= (título), ?genero=, ?precioMax=, ?teatroId=
  • Teatros: ?q= (nombre/barrio)
  • Usuarios: ?q= (nombre/email), ?rol=usuario|admin
